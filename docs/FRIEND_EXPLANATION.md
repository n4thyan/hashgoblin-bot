# Simple explanation

HashGoblin is a Discord economy bot where people gamble fake server currency called Glory.

The twist is that every result is made from SHA-256. When someone uses `/coinflip`, `/wheelspin`, `/lotto` or `/hashjackpot`, the bot saves a proof ID.

That proof shows the hash, the roll, the odds and how the result was picked. So it works like a casino bot, but every roll has receipts.

No real money. No cashouts. No crypto wallets. Just server points and goblin maths.
